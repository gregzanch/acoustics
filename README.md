# Acoustics

`acoustics` is a javascript library that provides useful functionality for acousticians.

## Usage

Installation via npm/yarn:

```bash
npm install acoustics
# or
yarn add acoustics
```

then import

```javascript
import AC from 'acoustics'

console.log(AC)
```
